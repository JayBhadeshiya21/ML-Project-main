Clinical Inputs Used for Prediction

The model evaluates objective measurements, examination values, and lifestyle indicators to estimate cardiovascular risk.

Objective Features

Age, Height, Weight, Gender

Examination Values

Lifestyle Indicators

Smoking, Alcohol, Physical Activity

Age

Captured in days and converted to years for display and model input.

Height

Patient height in centimetres. Combined with weight to compute BMI.

Weight

Patient weight in kilograms. Used alongside height for BMI derivation.

Gender

Encoded binary category — 1 for female, 2 for male.

Systolic Pressure

Arterial pressure during cardiac contraction (ap_hi). Key hypertension indicator.

Diastolic Pressure

Arterial pressure between heartbeats (ap_lo) — the resting pressure reading.

Cholesterol

Ordinal scale: 1 = normal, 2 = above normal, 3 = well above normal.

Glucose

Blood glucose level: 1 = normal, 2 = above normal, 3 = well above normal.

Smoking

Binary indicator for current tobacco use — 0 = non-smoker, 1 = smoker.

Alcohol

Binary indicator for regular alcohol consumption habits.

Activity

Binary indicator — 0 = sedentary lifestyle, 1 = regularly physically active.

Data Source Note

Model trained on the public Cardiovascular Disease dataset. Subjective lifestyle features are self-reported and may introduce noise. The model achieves approximately 73% accuracy on holdout data and is intended for educational use only.
